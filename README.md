
# ForensicVideo-FA - специализированное ПО для определения подлинности, выявления подделок, аномалий и искажений в видеопотоке

# Stack:
  - C++ 20v - язык программирования
  - Qt - фреймворк для создания кроссплатформенного GUI-приложения
  - OpenGl - фреймворк для визуализации сложных 2D/3D-объектов
  - CUDA - фреймворк/технология для параллельных вычислений на ядрах GPU
  - FFTW - библиотека для быстрого выполнения преообразования Фурье и анализа частотных областей
